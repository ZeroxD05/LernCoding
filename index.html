LernCoding zum Startbildschirm hinzufügen

    Auf Apple Geräten

  1. Öffne diese Webseite in Safari (nicht Chrome).
  2. Tippe unten auf das Teilen-Symbol.
  3. Scrolle nach unten und tippe auf „Zum Home-Bildschirm“.
  4. Wähle einen Namen (z. B. LernCoding).
  5. Tippe auf „Hinzufügen“.

    Auf Android Geräten

  1. Chrome öffnen.
  2. Auf das Drei-Punkte-Menü oben rechts tippen.
  3. Auf „Zum Startbildschirm hinzufügen“ tippen.
  4. Einen Namen wählen (z. B. LernCoding).
  5. Auf „Hinzufügen“ tippen.

Die App erscheint jetzt wie eine echte App auf deinem Gerät!
