Kompletter Fahrplan: Von Idee bis Live-Website

Dieser Kurs fuehrt dich Schritt fuer Schritt durch einen echten Website-Launch: Planung, Entwicklung, Animationen, GitHub, GitHub Pages und Vercel.

01

Projekt planen und Struktur aufsetzen

Du definierst Ziel, Seitenstruktur und Design-System. Danach legst du ein sauberes Projektgeruest mit index.html, styles.css und script.js an.

02

HTML-Grundgeruest professionell bauen

Semantischer Aufbau mit Header, Main, Sektionen und Footer. Du lernst saubere Struktur, Accessibility-Basics und wiederverwendbare Komponenten.

03

Modernes CSS und responsives Layout

Farbsystem, Typografie, Spacing und mobile-first Breakpoints. Ergebnis: gleiche Seite auf Smartphone, Tablet und Desktop ohne Brueche.

04

Animationen mit CSS und JavaScript

Scroll-Reveals, Hover-States, sanfte Transitions und kleine Interaktionen mit Mehrwert. Du lernst, wie Animationen elegant wirken statt zu stoeren.

05

Git und GitHub richtig nutzen

Repository erstellen, Commit-Strategie, Branching-Grundlagen und saubere Deploy-Commits. Damit hast du Versionierung wie in echten Teams.

06

Erster Live-Gang mit GitHub Pages

Du veroeffentlichst deine statische Seite ueber GitHub Pages und lernst, wie du eine URL teilst, Fehler analysierst und Updates live ausrollst.

07

Deployment mit Vercel fuer mehr Power

Projekt auf Vercel importieren, Domain verbinden, Umgebungsvariablen setzen und automatische Redeploys bei jedem Push aktivieren.

08

Launch-Checkliste und Wartung

Performance, SEO-Basics, Fallback-Seiten und Release-Routine. So bleibt deine Seite nicht nur online, sondern langfristig professionell.
